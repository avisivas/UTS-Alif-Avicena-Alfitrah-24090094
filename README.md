Deskripsi singkat

Project ini dibuat untuk memenuhi UTS saya di mata kuliah Pemograman Web 1 dengan komposisi sebagai berikut:

Halaman Login : User masuk menggunakan email dan password (password harus NIM), Validasi form dilakukan menggunakan JavaScript, Jika login berhasil, user diarahkan ke halaman dashboard.

Halaman Dashboard : Menampilkan ringkasan data seperti total produk, total penjualan, dan total pendapatan, Data diambil dari objek JavaScript (summary), Terdapat tombol untuk menuju halaman Data Produk.

Halaman Data Produk : Menampilkan tabel daftar produk beserta harga dan stok, Data di-load dari array JavaScript (products), Terdapat aksi Edit dan Delete (simulasi dengan alert & penghapusan data dari array).

Tombol Logout : Tombol logout di Dashboard dan Data Produk mengarahkan kembali ke halaman login.

Teknologi yang di gunakan :

HTML5 → struktur halaman

CSS3 → desain interface (tema merah marun/cream, responsive layout)

JavaScript murni → validasi login, render tabel, aksi edit/delete, navigasi halaman
